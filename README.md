# 🏨 Hotel Reservation System (Java Swing)

A simple Hotel Reservation System developed using **Java Swing**. The application allows users to view available rooms, book rooms, cancel reservations, and view booking details through a graphical user interface.

---

## 📌 Features

- 🏨 View Available Rooms
- 📅 Book Hotel Rooms
- ❌ Cancel Reservations
- 📋 View Booking Details
- 💳 Payment Mode Selection (Cash / Card / UPI)
- 🖥️ Java Swing GUI
- 📊 JTable for Room Display
- ☕ Object-Oriented Programming Concepts

---

## 🛠️ Technologies Used

- Java
- Java Swing
- JTable
- Event Handling
- OOP Concepts

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Hotel-Reservation-System-Java.git
```

2. Open the project in VS Code, Eclipse, or IntelliJ IDEA.

3. Compile

```bash
javac HotelGUI.java
```

4. Run

```bash
java HotelGUI
```
